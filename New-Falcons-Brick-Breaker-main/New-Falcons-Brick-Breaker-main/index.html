BRICK-BREAKER
WOULD YOU BREAK IT !
How To Play :
  1. 1. The objective is to break the bricks that are distributed around the top of the game screen.
  2. 2. Move your mouse to control the paddle.
  3. 3. The player gets 3 lives to start with; a life is lost if the ball hits the bottom of the screen.
  4. 4. If all lives are lost, the game is over.
